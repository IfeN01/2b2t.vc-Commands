# 2b2t.vc-Core

A Meteor Client addon for viewing 2b2t player stats.  
Supports versions **1.21.8** and **1.21.4**.

---

## Commands

| Command | Aliases | Description | Example |
|---------|--------|-------------|---------|
| `.chats2b2t` | `.chats` | Show recent chat messages | `.chats playerName`<br>`.chats playerName keyword` |
| `.connections2b2t` | `.conns` | Show JOIN/LEAVE connections | `.conns playerName` |
| `.deaths2b2t` | `.deaths` | Show recent deaths | `.deaths playerName`<br>`.deaths playerName 10` |
| `.firstseen2b2t` | `.fs` | Show first seen info | `.fs playerName` |
| `.kills2b2t` | `.kills` | Show recent kills | `.kills playerName`<br>`.kills playerName 10` |
| `.lastseen2b2t` | `.seen` | Show last seen info | `.seen playerName` |
| `.playtime2b2t` | `.pt` | Show playtime | `.pt playerName`<br>`.pt top month`<br>`.pt top all` |
| `.stats2b2t` | `.stats` | Show full stats | `.stats playerName` |

---

All outputs include **colored text** for readability.
